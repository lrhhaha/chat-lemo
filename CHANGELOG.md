# 更新日志

本文档记录 LangGraph Chat App 的所有版本更新和功能变更。

## [v0.1.0] - 2025-12-06

### ✨ 核心功能

#### 界面与交互

- 🎨 **赛博朋克主题界面** - 完整的视觉风格重构，科技感十足的 UI 设计
- 🌐 **全面中文化** - 界面文案、提示信息完全本地化
- 💬 **流式响应** - 实时 AI 对话体验，逐字输出响应内容
- 📝 **Markdown 渲染** - 支持表格、代码高亮、列表等丰富格式

#### 会话管理

- 🗂️ **多会话支持** - 创建、切换、删除多个独立聊天会话
- 💾 **自动保存** - 聊天记录自动持久化到 SQLite 数据库
- 📜 **历史记录** - 快速加载历史对话，支持会话恢复
- ✏️ **会话重命名** - 自定义会话名称，方便管理

#### 技术架构

- ⚡ **Next.js 16 + React 19** - 使用最新前端技术栈
- 🎣 **自定义 Hooks** - 模块化状态管理（useChatMessages、useSessionManager 等）
- 🔧 **组件化设计** - 可复用的 UI 组件库
- 🤖 **LangGraphJS 集成** - 强大的 AI Agent 编排能力
- 🗄️ **SQLite 数据库** - 轻量级本地数据持久化

- 开发环境热重载支持
